# LinkedIn Outreach Templates

Maturity: beta
Last updated: 2026-04-27

## Personalization rules

- Reference something specific from their profile, post, company, or workflow.
- Keep under 150 words.
- Lead with a useful observation, not a pitch.
- Do not automate generic connection messages.
- Do not use tool-list positioning unless they explicitly mention that tool.

---

## Template A — Warm intro after engagement

Hi [NAME],

Thanks for [engaging with my post about X / following me].

Your point about [specific thing] stood out because it usually points to a workflow issue, not just a tools issue.

I help founder-led teams clean up intake, follow-up, reporting, and handoffs with simple web/automation/AI systems.

If useful, I can share a quick 3-point teardown of where the workflow could be simplified.

Safeer

## Content prompt templates

### Post type A — workflow teardown

```text
Most teams do not have a tools problem.
They have a workflow visibility problem.

Example:
[describe messy before state]

The cleaner version:
1. [trigger]
2. [owner/decision]
3. [automation or internal tool]
4. [output/report]
5. [failure/edge-case handling]

The lesson: automate after the workflow is clear, not before.
```

### Post type B — practical insight

```text
A useful automation system has 5 parts:

1. a clear trigger
2. clean input data
3. one obvious owner
4. a visible output
5. an error path

Most broken automations are missing #3 or #5.
```

### Post type C — founder/operator observation

```text
If follow-up depends on memory, it will eventually fail.

The fix is not “be more disciplined.”

The fix is a system where:
- the trigger is captured
- the next step is assigned
- the status is visible
- the fallback is obvious

That is the kind of operational leverage small teams actually feel.
```

## Key rules

### Do

- Reference something specific.
- Keep it short.
- Talk about the workflow/business problem first.
- Offer a teardown or useful next step.
- Follow up 3-5 days later if appropriate.

### Don’t

- Send generic connection automation.
- Lead with “I help X” in every message.
- Overuse tool names.
- Claim results you cannot prove.
- Send more than 2 follow-ups total.
