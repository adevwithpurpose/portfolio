# Client Hunting Research — 2026-03-30

## Channels Summary

### 1. Upwork
- 1,178+ n8n jobs on Upwork
- AI Agent automation jobs paying $1K-$10K+
- Agentic AI Automation Expert roles available
- **Strategy:** Only bid on $1K+. Write store-specific proposals. Link to portfolio.

### 2. n8n Community Forum
- `community.n8n.io` — has "Looking for freelancers" section
- Clients already know n8n = high intent
- **Strategy:** Build reputation by answering questions. Post in jobs section.

### 3. n8n Jobs Board
- `n8ndevs.com/jobs` — dedicated n8n job board
- **Strategy:** Check daily. Apply to relevant posts.

### 4. LinkedIn
- Groups: AI Automation, Business Process Automation, n8n Automation, Marketing Automation
- Decision-makers: Operations Managers, CTOs, Founders
- **Strategy:** Post case studies. DM store owners. Use Sales Navigator.

### 5. X / Twitter
- Monitor: #n8n, #automation, #AIagent, #Make.com, #Zapier
- Phrases: "need to automate", "looking for n8n dev", "workflow killing me"
- **Strategy:** Post tips/case studies. Engage with hashtag content. DM people asking for help.

### 6. Reddit
- `r/n8n` — n8n help requests, freelancer searches
- `r/automation` — broad automation
- `r/artificialintelligence` — AI agent interest
- `r/entrepreneur` / `r/smallbusiness` — SMB owners
- `r/forhire` — post your services
- `r/Shopify` — e-commerce owners
- **Strategy:** Answer questions genuinely. Post in r/forhire monthly.

### 7. Specialized Platforms
- `agent.ai` — AI agent marketplace
- `contra.com` — n8n + automation freelancers
- `freelancer.com` — n8n projects
- **Strategy:** List services, check regularly.

### 8. Shopify Ecosystem
- Shopify Partners Directory — get listed
- `r/Shopify` community
- Storetasker — Shopify expert marketplace
- **Strategy:** Get on Partners Directory. High-intent clients.
