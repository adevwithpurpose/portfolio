# Playbook — Client Discovery Prep

Maturity: beta

## Goal

Prepare quickly for a prospect/client call without over-researching.

## Inputs

- prospect/company name
- website/social links
- problem statement, if available
- previous messages

## Steps

1. Search existing Obsidian/Honcho context first.
2. Review the prospect website and public positioning.
3. Identify likely friction:
   - slow/manual follow-up
   - weak landing pages
   - scattered tools
   - unclear reporting
   - missing automations
4. Prepare 5 focused questions.
5. Prepare 2-3 possible quick-win ideas.
6. Prepare a calm next-step offer: workflow teardown, landing page fix, automation build, or internal tool.

## Output

- short company summary
- likely pain points
- discovery questions
- proposed next step
- risks/sensitive notes
